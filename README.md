# Xylo: Game-Core

Android game optimization module for AxManager. Auto-switches between Gaming Mode and Battery Saver Mode based on game detection.

## Features

- Auto game detection
- Dynamic refresh rate (60Hz/90Hz/120Hz/144Hz)
- GPU/HWUI rendering tweaks (SkiaGL / SkiaVK)
- Thermal & power optimization
- RAM compaction
- Network optimization
- GMS doze
- WebUI control panel
- Auto Clear PowerKeeper (Xiaomi only)

## Download

Install via AxManager using plugin ID: `14700`

## Requirements

- Android 11+
- AxManager installed

## WebUI

Access the WebUI through AxManager to control all settings:
- Game list management
- Utility toggles
- Renderer engine
- Refresh rate
- DNS private
- View logs
